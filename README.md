# Examples

## Normal
```
python main.py -p ullist "Return a list of random human names"
```

## Segmented
```
python main.py -p bullets -O segment=2000 -u https://www.ub.gu.se/sv/lana-och-logga-in/lana-pa-biblioteket
```

### Without segments:
```
- Låna böcker direkt via internet
- Användarnamn och lösenord
- Hämta respektive bok i en timme innan ni tar examen
- Försenade återlämnande kostar pengar
- Skulle finnas förlorat material betalar man en ersättningskostnad
```

### With segments:
```
- Gothenburg University Library provides books and resources to students and faculty members aged 15+.
- Students can borrow books with their university cards while others require borrower's cards.
- Searching for books can be done via SuperSearch at various libraries.
- E-books can be ordered through SuperSearch.
- Distance learners can receive delivery services for free on e-book orders.
- Overdue book fines cost SEK 10 per day.
- Lost materials result in a fixed fee of SEK 600.
- Academic resources and communication channels can be accessed online through Canvas, Medarbetarportalen, and Gothenburg University.
```
